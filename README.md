# lw

This repository collects planning notes and starter ECG algorithm projects for preparing an algorithm-focused PhD direction in biomedical instrumentation and vital-sign intelligent sensing.

## Contents

- `task_plan.md`: main PhD execution plan.
- `roadmap_36_months.md`: 36-month milestone roadmap.
- `reading_and_reproduction_list.md`: literature and reproduction checklist.
- `weekly_execution_template.md`: weekly planning template.
- `findings.md`: public-source research notes and direction judgment.
- `progress.md`: local execution log.
- `ecg-dl-baseline/`: MIT-BIH ECG deep-learning baseline project.
- `ecg-wearable-fm-benchmark/`: PTB-XL wearable robustness and foundation-model benchmark starter project.

## Notes

Local datasets, virtual environments, model weights, and experiment outputs are intentionally ignored by Git. Recreate them by following the README and run notes inside each project directory.
