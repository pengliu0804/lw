# 文献与复现清单

## 文献线 1：心震图与机械心动信号

目标：理解 SCG/GCG 的波形来源、心动周期事件、噪声来源和临床应用。

阅读时记录：

- 传感器位置和采样率。
- ECG 同步方式。
- AO、AC、MC 等事件定义。
- 预处理和滤波参数。
- 标签来源：超声、导管、人工标注或算法标注。
- 验证方式：受试者级、窗级、跨设备、跨场景。

可复现任务：

- SCG/GCG 心拍分割。
- 机械心动事件定位。
- 运动伪迹检测与去噪。
- 瓣膜病或心衰分类。

## 文献线 2：非接触生命体征雷达

目标：理解 FMCW/毫米波雷达心肺信号提取、目标状态检测和实时性约束。

阅读时记录：

- 雷达参数、距离门选择、相位解缠。
- 呼吸与心跳频段分离。
- 静止、微动、体动、多人场景处理。
- 实时算法复杂度。
- 和接触式 ECG/PPG/呼吸带的对齐方式。

可复现任务：

- 雷达呼吸率和心率估计。
- 目标状态分类。
- 体动干扰检测。
- 雷达与 ECG/PPG 融合。

## 文献线 3：医学时序深度学习

目标：把深度学习优势迁移到生理时序，而不是停留在通用模型调参。

阅读时记录：

- 输入粒度：点、心拍、窗口、受试者、就诊。
- 标签粒度：窗口标签是否来自受试者标签弱传播。
- 模型：1D-CNN、TCN、RNN、Transformer、masked model、contrastive learning。
- 泛化设置：跨受试者、跨医院、跨设备、跨疾病阶段。
- 可解释方法：saliency、attention、prototype、feature attribution、错例分析。

可复现任务：

- 1D-CNN/TCN/Transformer 分类基线。
- 对比学习预训练后少样本微调。
- masked reconstruction 预训练。
- domain adaptation 或 test-time adaptation。

## 复现优先级

| 优先级 | 方法 | 用途 |
|---|---|---|
| P0 | 传统滤波 + 峰值检测 | 建立可解释下限和错误类型 |
| P0 | CWT/STFT + XGBoost | 对齐老师代表工作中的时频和传统 ML 思路 |
| P0 | 1D-CNN | 简单深度学习基线 |
| P1 | TCN/Transformer | 时序长依赖和多尺度建模 |
| P1 | Contrastive learning | 小样本和弱标签场景 |
| P2 | Domain adaptation | 跨设备/跨场景泛化 |
| P2 | Uncertainty calibration | 临床可信输出和质量控制 |
